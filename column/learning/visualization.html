echarts 折线图绘制区域渐变色 ​

通过 series 里面的 areaStyle 属性来配置

js
series: [
  {
    areaStyle: {
      color: {
        type: "linear",
        x: 0,
        y: 0,
        x2: 0,
        y2: 1,
        colorStops: [
          {
            offset: 0,
            color: "rgba(0,162,255,0.3)", // 0% 处的颜色
          },
          {
            offset: 1,
            color: "rgba(0,162,255,0)", // 100% 处的颜色
          },
        ],
        global: false,
      },
    },
  },
];

echarts 折线图中展示平均值虚线 ​

通过 series 里面的 markLine 属性来配置

js
series: [
  {
    markLine: {
      symbol: ["none", "none"], // none为标线两端的样式为无，可更改
      data: [
        {
          silent: false,
          yAxis: 50, // 警戒线位置，这个赋值为纵轴50
          label: {
            position: "insideEndBottom", // 文字位置
            distance: [-40, 10],
            fontSize: 12,
            formatter: (params) => {
              return `月稼动率\n${params.value}%`;
            }, //文字
            color: "#F8931F", // 文字颜色
            fontWeight: "bold",
          },
          lineStyle: { type: "dashed", color: "#F8931F", width: 1 }, // 样式： 线型、颜色、线宽
        },
      ],
    },
  },
];

echarts 中的悬浮框 ​

通过最外层的 tooltip 属性来设置:tooltip 中如果 trigger 的值未设置或者为'item'，那么 formatter 中的参数为每一项;trigger 的值为'axis',formatter 中的参数为整个数据数组

js
// 第一种情况
 tooltip: {
    formatter: (params) => {
      return `${params.seriesName}:${params.data.toFixed(2)}%`;
    },
 }
// 第二种情况
tooltip: {
  trigger: "axis",
  axisPointer: {
    type: "shadow",
  },
  formatter: function (params) {
    console.log(params, 'params+++');
    let tip = "";
    tip += params[0].name + "<br />";
    params.map((i) => {
      const newValue = i.data.label || i.data+'%'
      tip +=
        `<span style="display:inline-block;margin-right:5px;border-radius:10px;width:10px;height:10px;background-color:${i.color};"></span>` +
        i.seriesName +
        "：" +
        newValue +
        "<br />";
    });
    return tip;
  }
},

echarts 中折线图的图例 ​

图例通过最外层的 legend 属性配置:如果是空心圆左右各为一条线，即为折线图默认图例，此时可以分别调整图例中线的颜色和空心圆的颜色

js
legend: {
  data: [
    {
      name: "xxx",
      itemStyle: {
        color: "#000",
        fontSize: 16,
      },
      lineStyle: {
        color: "#F00",
      },
    },
  ];
}

如果是实心圆左右各为一条线,则需要通过 svg 的矢量路径 path 属性定义新的图例样式，此时只能通过实心圆的颜色来调整图例整体的颜色，图例线的颜色配置不生效(svg 图片可以找 UI 制作，然后打开取 path 属性即可，若存在两个 path，则中间用空格拼接)

js
legend: {
  data: [
    {
      icon: "path://M0,4.33H40v2H0Z M20,0a5,5,0,1,1-5,5A5,5,0,0,1,20,0Z",
      name: "xxx",
      itemStyle: {
        color: "#000",
        fontSize: 16,
      },
    },
  ];
}

        )}</div>
